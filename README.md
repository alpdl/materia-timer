# Materia Timer

Калькулятор генерации **материи** и фарма **нейтрония** для AFK-сессий в Minecraft (мод-сборки уровня GregTech: New Horizons и подобных).

Чистый клиент, никакого бэка — деплоится на GitHub Pages.

## Возможности

- **Материя.** Считает производительность в тик / секунду / минуту / час из заданной энергии (FT) и TPS.
- **Нейтроний.** Считает производительность набора сборщиков (от 1 до 60 штук, четырёх типов) в кучках, кусочках, слитках и блоках — в минуту, час и день.
- Параметры сохраняются в `localStorage`, текущая вкладка — в hash URL (`#materia` / `#neutronium`).
- Тёмная тема со стеклом, без сторонних UI-фреймворков.

## Формулы

### Материя

```
материя_в_тик = (энергия_в_миллионах × 1_000_000) / 40_000_000
                = энергия_в_миллионах / 40
```

То есть **40 миллионов FT** при TPS=20 → ровно **1 материя/тик** = 20 материи/сек.

### Нейтроний

Базовый цикл сборщика — **7 минут**. Иерархия:

```
1 блок  = 9 слитков  = 81 кусочек = 729 кучек
```

Сборщик каждого типа за цикл производит **одну единицу своего номинала** (одну кучку / кусочек / слиток / блок). Итог приводится к кусочкам и пересчитывается во все остальные единицы:

```
кусочки_в_день = циклы_в_день × (кучки/9 + кусочки + слитки×9 + блоки×81)
```

## Запуск

```bash
npm install
npm run dev          # http://localhost:5173
npm run build        # сборка в ./dist
npm run preview      # локальный предпросмотр прод-сборки
npm test             # vitest, прогон один раз
npm run test:watch   # vitest в watch-режиме
npm run typecheck    # tsc -b --noEmit
npm run lint         # eslint
npm run format       # prettier --write
```

## Деплой

Пуш в `main` → GitHub Actions собирает проект и публикует `dist/` на GitHub Pages (workflow в `.github/workflows/deploy.yml`).

В `vite.config.ts` указан `base: "/materia-timer/"` для production-сборки — он совпадает с именем репозитория. Если форкнешь под другим именем — поправь.

После первого пуша зайди в **Settings → Pages → Build and deployment → Source** и выбери **GitHub Actions** — иначе workflow не сможет опубликовать сайт.
